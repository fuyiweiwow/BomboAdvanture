# 武道会模式设计

## 概述
模拟《龙珠》天下第一武道会风格，玩家与各路强者进行 QQ 堂原汁原味的炸弹对战。

> **状态：设计阶段，未实现**

## 核心流程

### 1. 参赛准备
- 进入武道会前，玩家可以调整自身素质：
  - **基础威力**（Power）：炸弹爆炸范围
  - **速度**（Speed）：移动速度
  - **可放置炸弹数**（Bomb）：同时存在的上限
  - **初始血量**：固定 5 格血
- 属性有数值限制（不能无限堆），通过消耗金币购买属性点

### 2. 海选轮
- 海选赛场中有**可炸方块**（破坏后可获得基础资源）
- 玩家通过炸方块收集金币/道具，积累初始优势
- 海选目标：击败所有对手晋级

### 3. 淘汰赛制
```
[海选]
   │
   ▼
[32强] ── 1v1 淘汰
   │
   ▼
[16强]
   │
   ▼
[8强]
   │
   ▼
[半决赛]
   │
   ▼
[决赛]
```

### 4. 对阵图
- 类似 **Little Fighter 2** 的对阵图风格
- 左侧显示所有选手头像/名称
- 晋级路线用连线表示
- 当前轮次高亮

### 5. 战斗规则
- 每局 1v1，在封闭擂台中进行
- 初始 5 格血，每次被炸到扣 1 格
- 血量归零 → 进入**濒死状态**（无法移动/放置炸弹，等待救援）
- **场间休息**：玩家可以在商店用金币购买**仙豆**
  - 仙豆效果：将濒死状态的玩家恢复 1 格血
  - 每场比赛限购有限数量

### 6. 对手系统
- 对手从游戏世界观中的知名 NPC 中抽取（怪物、Boss、冒险者）
- 每轮对手逐渐变强：
  - 初期：普通怪物（史莱姆、蝙蝠等）
  - 中期：精英怪物 + 特定技能
  - 后期：Boss 级对手 + 特殊能力
  - 决赛：神秘强者

### 7. 奖励
- 每晋级一轮获得金币奖励
- 冠军获得额外奖励（限定道具/称号）

## 技术需求
- 独立于冒险模式的战斗场景
- 1v1 擂台地图（比标准地图小，无探索元素）
- 对手 AI 复用现有 NPC 系统，可配置加强
- 对阵图 UI 系统
- 金币经济系统与冒险模式共享
- 仙豆特殊道具（可在濒死时使用）
