# Bombo Adventure — 开发计划与设计文档

## 一、项目定位

以**城市为中转舞台**的 **Roguelike RPG** 冒险游戏。

**核心战斗**：炸弹即一切。不提供除炸弹外的任何武器，所有道具、技能、成长系统均围绕炸弹玩法设计。

游戏模式分为：

| 模式 | 说明 |
|------|------|
| **Rogue 冒险模式** | 以城市为中转舞台，接取任务进入随机/固定关卡探索，获取道具和金币，强化自身 |
| **武道会模式** | 炸弹人之间正面对决，比拼炸弹技巧与道具策略，用于赚取金币 |

## 二、阶段目标

### 阶段一：核心玩法循环

玩家以城市为据点，在以下设施中成长：

| 设施 | 功能 |
|------|------|
| **冒险工会** | 接取随机刷新的任务（根据冒险者等级），完成任务获得奖励并提升等级 |
| **道具店** | 购买/出售冒险道具（药水、护甲、钥匙等，均为炸弹玩法服务） |
| **武道会** | 参与炸弹人对决赚取金币 |

#### 核心战斗机制
- **唯一攻击手段是炸弹**（放置、踢、连锁引爆等）
- 道具全部围绕炸弹设计：炸弹数量、爆炸范围、移动速度、防御、回血、钥匙开锁等
- 无刀剑、弓箭等传统武器

#### 关卡系统
- 任务地点为各种场景（关卡），包含：
  - **固定地图** — 手工设计的关卡
  - **程序生成地图** — 每次进入随机生成，支持 Roguelike 玩法
- 关卡掉落：
  - 冒险道具（药水、护甲、钥匙等）
  - 药水制作材料（为阶段二炼药系统做准备）

### 阶段二：炼药系统
- 利用关卡中收集的材料制作药水（规划中）

## 四、技术栈

- **引擎**: Godot 4.6.2 Mono
- **语言**: GDScript
- **组件系统**: 基于 Atlas 的部位独立换装（body / face / cloth / hair / leg / cap / eye / ear / mouth / cladorn / fpack / npack / thadorn / footprint）
- **数据格式**: JSON（角色、地图、关卡、帧动画）
